# Dossier Professionnel – TSSR

Ce repository contient mon **dossier professionnel réalisé dans le cadre de la formation Technicien Supérieur Systèmes et Réseaux (TSSR)**.

## Objectif

Présenter les compétences acquises en administration systèmes, réseaux et cybersécurité.

## Technologies utilisées

- Windows Server
- Active Directory
- Linux
- DNS / DHCP
- VMware / VirtualBox
- Wireshark
- Nmap
- Metasploit
- Kali Linux
- pfSense
- DualHoming AD/serveur mail en environnement virtualisé
